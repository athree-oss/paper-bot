# Contributing

Thank you for your interest in contributing to paper-bot!

## Getting started

1. Fork the repository and clone your fork.
2. Install dependencies: `pnpm install`
3. Create a new branch for your change: `git checkout -b my-feature`

## Making changes

- Keep changes focused and minimal.
- Follow the existing code style (enforced by `.editorconfig` and `.prettierrc`).
- Add or update tests when relevant.

## Submitting a pull request

1. Push your branch to your fork.
2. Open a pull request against the `main` branch of this repository.
3. Describe what the change does and why.

## Reporting issues

Open an issue on GitHub with a clear description of the problem, steps to reproduce, and any relevant context.
